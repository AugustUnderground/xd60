# POK3R Style 60% Layout

Based on `crd_ansi` and GK61 Layout.

## Base Layer Layout

```
,-----------------------------------------------------------------------------------------.
| Esc |  1  |  2  |  3  |  4  |  5  |  6  |  7  |  8  |  9  |  0  |  -  |  =  | Backspace |
|-----------------------------------------------------------------------------------------+
|  Tab   |  Q  |  W  |  E  |  R  |  T  |  Y  |  U  |  I  |  O  |  P  |  [  |  ]  |   \    |
|-----------------------------------------------------------------------------------------+
| Esc/Ctrl|  A  |  S  |  D  |  F  |  G  |  H  |  J  |  K  |  L  |  ;  |  '  |    Enter    |
|-----------------------------------------------------------------------------------------+
|   Shift   |  Z  |  X  |  C  |  V  |  B  |  N  |  M  |  ,  |  .  |  ?  |      Shift      |
|-----------------------------------------------------------------------------------------+
| Ctrl | Super |  Alt  |               Space               | AltGr | Super | Crrl  |  Fn  |
`-----------------------------------------------------------------------------------------'
```

## Function Layer Layout

```
,-----------------------------------------------------------------------------------------.
|  `  | F1  | F2  | F3  | F4  | F5  | F6  | F7  | F8  | F9  | F10 | F11 | F12 |    Del    |
|-----------------------------------------------------------------------------------------+
|        |     |     |     |     |     |     |  PS |  SL |  PB |     |     |     |        |
|-----------------------------------------------------------------------------------------+
|         |     |     |     |     |     |     | Ins |  HM |  PU |     |     |             |
|-----------------------------------------------------------------------------------------+
|           |     |     |     |     |     |     | Del | End | PD  |     |                 |
|-----------------------------------------------------------------------------------------+
|RGBCyc| Brt+  | Brt-  |            RGB Toggle             | Left  | Down  | Right |  Fn  |
`-----------------------------------------------------------------------------------------'
```

## Build

To build the default keymap, simply run:

```bash
$ make xd60/rev2:augustunderground  # XD60 rev2

$ make xd60/rev3:augustunderground  # XD60 rev3
```
